# Dev Books Printer 

Показывает новинки технических издательств

Список издательств:
* дмк-пресс
* издательство питер
* bhv

## Как использовать
* требуется python 3.5
* установить модули:
```bash
$ pip install -r requirements.txt
```
* Запустить:

```bash
$ python books.py
```

### Цели
Код написан в учебных целях и ради интереса
